# What Remains

The official repo for the source code of What Remains, a Global Game Jam 2026 submission made in under 48 hours.

## Dependencies

### Wwise UE Plugin
Built from the Wwise 2024.1.1.8691 SDK for Unreal Engine 5.4.4. The plugin is packaged with the game executable upon export.
Can be built with the Wwise Launcher using the aforementioned SDK.
